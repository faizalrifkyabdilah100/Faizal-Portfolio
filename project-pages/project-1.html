EVENTA — Event Management Platform

Laravel 12 React + Inertia QR Check-In Tailwind CSS

Deskripsi Proyek

EVENTA adalah platform manajemen acara end-to-end yang dirancang untuk menangani seluruh siklus acara, mulai dari pembuatan event, pendaftaran peserta, hingga sistem verifikasi di lokasi (on-site) secara real-time.

1. Manajemen Acara & Full-stack Architecture

Menggunakan Laravel 12 sebagai core engine dan React + Inertia untuk pengalaman user interface yang seamless.

  • Implementasi CRUD acara yang dinamis dengan dukungan multi-sesi (EventSession).
  • Kustomisasi tema acara (logo dan warna) sesuai kebutuhan penyelenggara.
  • Alur pendaftaran terstruktur dengan manajemen penyimpanan data peserta per-sesi.

2. Ticketing & QR Check-In System

Sistem verifikasi canggih untuk meminimalkan antrean dan meningkatkan keamanan tiket.

  • Pembuatan tiket digital otomatis dengan kode registrasi unik dan QR Code.
  • Sistem pengiriman tiket melalui endpoint sendTicket.
  • Mesin verifikasi QR real-time pada sisi Admin untuk update status kehadiran (Attendance) secara instan.

3. Dashboard Admin, Analytics & Keamanan

Pusat kontrol yang memberikan wawasan data kepada penyelenggara acara.

  • Dashboard analitik untuk memantau statistik pendaftaran dan attendance rate per acara.
  • Otentikasi dan verifikasi berlapis menggunakan Laravel Fortify.
  • Proteksi API menggunakan Laravel Sanctum untuk keamanan data transaksi.
